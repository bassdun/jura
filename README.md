# Jura Live Theme

Shopify theme for topartner.myshopify.com

## Theme Information

- **Live Theme**: Copy of Vessel Implementation staging
- **Store**: topartner.myshopify.com
- **Exported**: 17 November 2025

## Development

To work with this theme locally:

```bash
shopify theme dev --store topartner.myshopify.com
```

To push changes to Shopify:

```bash
shopify theme push --store topartner.myshopify.com
```

## Important Notes

- All settings and banners are stored in `config/settings_data.json`
- Always commit changes before pushing to Shopify
- Use branches for testing major changes
